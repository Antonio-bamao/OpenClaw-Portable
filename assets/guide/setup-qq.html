QQ Bot 接入帮助

本页用于配置 QQ Bot 的 AppID 和 AppSecret。当前流程是“填写凭据 -> 检查配置 -> 启用 QQ”。

1. 准备凭据

  1. 登录 QQ 开放平台并创建机器人。
  2. 记录该机器人的 AppID 和 AppSecret。
  3. 回到启动器，在“QQ Bot”卡片里填入这两个值并保存。

2. 检查配置

  1. 点击 检查 QQ 配置。
  2. 若状态变成“待启用”，说明凭据格式和当前便携包的 QQ 运行时支持都已通过本地检查。
  3. 若看到“缺少内置扩展”，说明当前便携包里的 OpenClaw runtime 缺少 qqbot 扩展，需要重新构建或更新程序包。

3. 启用 QQ

  1. 当卡片显示“待启用”时，点击 启用 QQ。
  2. 启用后，QQ 私聊、群聊和富媒体消息可以由 OpenClaw runtime 接管。
  3. 若暂时不用，可点击 停用。

4. 常见问题

  • “未配置”：请先填写并保存 AppID / AppSecret。
  • “配置无效”：请重新检查凭据是否为空或是否填错。
  • “缺少内置扩展”：当前包缺少 QQ Bot runtime 插件，不是账号问题。
  • “待启用”：凭据检查已经通过，下一步是点击 启用 QQ。
