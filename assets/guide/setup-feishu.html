飞书私聊接入帮助

本页用于配置飞书企业自建应用，让用户私聊 Bot 时进入 OpenClaw，并把回复发回同一个飞书私聊会话。本轮只支持私聊，不支持群聊、@ 提及或群白名单。

1. 创建飞书应用

  1. 打开飞书开放平台，创建一个企业自建应用。
  2. 在应用凭证页复制 App ID 和 App Secret。
  3. 在 OpenClaw 启动器的“飞书私聊渠道”卡片中填入凭据并保存。

2. 配置机器人与事件

  1. 在飞书应用后台启用机器人能力。
  2. 按飞书后台提示完成事件订阅或长连接配置。
  3. 确保应用具备接收私聊消息和发送消息所需权限。
  4. 回到启动器点击“测试连接”，测试通过后点击“启用飞书私聊”。

3. 常见问题

  • “配置无效”：请重新检查 App ID 和 App Secret。
  • “连接失败”：请确认飞书后台机器人、事件订阅、权限已配置完整，并重启 OpenClaw 服务。
  • “待启用”：凭据已保存，但尚未写入运行时启用配置，请点击“启用飞书私聊”。
