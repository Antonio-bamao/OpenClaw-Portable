微信 ClawBot 接入帮助

本页用于把微信 ClawBot 渠道接到 OpenClaw。当前流程是“安装插件 -> 扫码登录 -> 启用微信”。扫码完成后，启动器会尽量自动把状态推进到“待启用”。

1. 安装插件

  1. 在启动器主界面的“微信 ClawBot”卡片点击 安装微信插件。
  2. 等待安装完成后，卡片状态应变成“待扫码”。
  3. 如果安装失败，请先检查网络、Node 运行时和 OpenClaw runtime 是否完整。

2. 扫码登录

  1. 点击 扫码登录，启动器会打开一个交互终端窗口。
  2. 按终端提示完成二维码扫码或登录确认。
  3. 扫码成功后，回到启动器主界面，状态通常会刷新为“待启用”。

3. 启用微信

  1. 当卡片显示“待启用”时，点击 启用微信。
  2. 启用后，微信私聊消息会被投递到 OpenClaw。
  3. 若需要停用该渠道，可点击 停用。

4. 常见问题

  • “未安装”：先点击 安装微信插件。
  • “待扫码”：插件已安装，下一步是点击 扫码登录。
  • “待启用”：扫码已完成，下一步是点击 启用微信。
  • “扫码失败”：请重试扫码登录，并保留终端窗口查看错误信息。
